GENERAL PROVISIONS The present Privacy Policy (hereinafter — "Policy") defines the procedure for processing personal data of Voltrix service users (hereinafter — "Service", "we", "us", "our"). By using our Service, you agree to the terms of this Privacy Policy. 2. WHAT DATA WE COLLECT 2.1 Personal user data: • Phone number — for registration, authentication and communication with user • Username (optional) — for service personalization • FCM tokens — for sending push notifications • Authentication data — JWT tokens, refresh tokens for security 2.2 Financial data: • Bank card data — masked card numbers, card types, last 4 digits • Card tokens (rectoken) — for secure payments through Flitt • Payment history — amounts, statuses, transaction descriptions • Deposit data — frozen and debited amounts 2.3 Service usage data: • Rental history — start/end times, pickup/return stations, cost • Station data — coordinates, addresses, online/offline status • Return attempts — information about powerbank return attempts • Debt and penalty system — debt amounts, penalties 2.4 Technical data: • IP addresses — for security and analytics • User-Agent — browser/application information • System logs — for monitoring and debugging • WebSocket connections — for real-time notifications 3. LEGAL BASIS FOR DATA PROCESSING Personal data processing is carried out on the basis of: • User consent — for registration and service use • Contract performance — for rental service provision • Legitimate interests — for security and fraud prevention • Legal obligations — for compliance with legislation 5. DATA TRANSFER TO THIRD PARTIES 5.1 Payment systems: • Flitt — for payment processing and card management • Data is transferred encrypted with PCI DSS compliance 5.2 SMS providers: • Twilio — for SMS verification code delivery • Only phone numbers and message texts are transferred 5.3 Technical services: • PostgreSQL — for data storage (encrypted connection) • Redis — for caching and sessions • Winston — for logging (without personal data) 6.
