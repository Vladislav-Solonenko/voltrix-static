Политика конфиденциальности

ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ Voltrix - Система аренды пауэрбанков Последнее обновление: 13.10.2025 1. ОБЩИЕ ПОЛОЖЕНИЯ Настоящая Политика конфиденциальности определяет порядок обработки персональных данных пользователей сервиса Voltrix. Используя наш Сервис, вы соглашаетесь с условиями настоящей Политики конфиденциальности. 2. КАКИЕ ДАННЫЕ МЫ СОБИРАЕМ 2.1 Персональные данные пользователей: • Номер телефона — для регистрации и связи с пользователем • Имя пользователя (опционально) — для персонализации сервиса • Токены уведомлений — для отправки push-уведомлений • Данные аутентификации — для обеспечения безопасности 2.2 Финансовые данные: • Данные банковских карт — замаскированные номера карт, типы карт • Токены карт — для безопасного проведения платежей • История платежей — суммы, статусы, описания транзакций • Данные депозитов — суммы заморозки и списания средств 2.3 Данные об использовании сервиса: • История аренды — время начала/окончания, станции, стоимость • Данные о станциях — координаты, адреса, статус • Попытки возврата — информация о попытках возврата пауэрбанков • Система долгов и штрафов — суммы задолженности, штрафов 2.4 Технические данные: • IP-адреса — для обеспечения безопасности • Информация о браузере/приложении • Логи системы — для мониторинга • Данные соединений — для уведомлений 3. ЦЕЛИ ОБРАБОТКИ ДАННЫХ Мы обрабатываем ваши персональные данные для следующих целей: 3.1 Основные функции сервиса: • Регистрация и аутентификация пользователей • Обеспечение работы системы аренды пауэрбанков • Обработка платежей и управление депозитами • Отправка уведомлений о статусе аренды • Предотвращение мошенничества и обеспечение безопасности 3.2 Улучшение сервиса: • Анализ использования для улучшения функциональности • Мониторинг производительности системы • Техническая поддержка пользователей 4. ПРАВОВЫЕ ОСНОВАНИЯ ОБРАБОТКИ Обработка персональных данных осуществляется на основании: • Согласия пользователя — для регистрации и использования сервиса • Исполнения договора — для предоставления услуг аренды • Законных интересов — для обеспечения безопасности и предотвращения мошенничества • Правовых обязательств — для соблюдения требований законодательства 5. ПЕРЕДАЧА ДАННЫХ ТРЕТЬИМ ЛИЦАМ 5.1 Платежные системы: • Специализированный сервис — для обработки платежей и управления картами • Данные передаются в зашифрованном виде с соблюдением стандартов безопасности 5.2 SMS-провайдеры: • Специализированный сервис — для отправки SMS-кодов верификации • Передаются только номера телефонов и тексты сообщений 5.3 Технические сервисы: • Специализированные сервисы — для хранения данных, кэширования и логирования • Все соединения зашифрованы 6. БЕЗОПАСНОСТЬ ДАННЫХ 6.1 Технические меры: • Шифрование данных при передаче • Хеширование паролей и токенов • Ограничение доступа к данным • Регулярные обновления безопасности 6.2 Организационные меры: • Обучение сотрудников по защите данных • Политики доступа и аудита • Регулярные проверки безопасности 7. СРОКИ ХРАНЕНИЯ ДАННЫХ • Активные пользователи — данные хранятся до удаления аккаунта • SMS-коды — 5 минут после создания • Логи — до 30 дней • Финансовые данные — согласно требованиям законодательства (до 5 лет) • Неактивные аккаунты — до 3 лет после последней активности 8. ПРАВА ПОЛЬЗОВАТЕЛЕЙ Вы имеете право: • Получать информацию о обработке ваших данных • Исправлять неточные данные • Удалять ваши данные (право на забвение) • Ограничивать обработку данных • Переносить данные в другой сервис • Отзывать согласие на обработку данных • Подавать жалобы в надзорные органы 9. COOKIES И АНАЛИТИКА Мы используем: • Сессионные cookies — для аутентификации • Технические cookies — для работы соединений • Аналитические данные — агрегированная статистика использования 10.
